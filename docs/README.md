# Documentacion

Este directorio concentra la documentacion final de la prueba tecnica.

- `docs/arquitectura.md`: descripcion del pipeline y decisiones tecnicas.
- `docs/diagramas.md`: diagramas Mermaid (pipeline y escalado).
- `docs/operacion.md`: guia de ejecucion local, Docker y Airflow.
- `docs/devops.md`: CI/CD, pre-commit, Makefile y comandos.
- `docs/cloud_scaling.md`: propuesta de escalado en cloud (conceptual).
